# Stock Portfolio Tracker

## Description
This project calculates the total investment value of a stock portfolio. Users enter the quantity of each stock they own, and the program calculates the overall investment value using predefined stock prices.

## Features
- Predefined stock prices
- User input for stock quantities
- Calculates total investment
- Optional result saving to a text file

## Technologies Used
- Python 3

## Concepts Used
- Dictionary
- For Loop
- User Input
- Arithmetic Operations
- File Handling

## How to Run
1. Install Python 3.
2. Download the project.
3. Run:# CodeAlpha_Stock
A beginner-friendly Python application that calculates the total value of a stock portfolio using predefined stock prices.
